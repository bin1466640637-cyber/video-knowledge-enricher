# Video Knowledge Enricher

Video notes enrichment skill for Hermes Agent.

## 功能

把视频链接（YouTube / 哔哩 / 抖音）转换成完整的 Obsidian 笔记，包含：
- 字幕/弹幕提取
- 关键画面截图
- 背景知识扩展搜索
- 与现有知识库的交叉引用
- 带时间戳的完整笔记

## 目录结构

```
video-knowledge-enricher/
├── SKILL.md                              # 主技能文件
└── references/
    ├── kazhig-style-guide.md # 公众号风格指南
    └── video-download-platforms.md       # 各平台下载方案
```

## 使用方法

在 Hermes Agent 中直接发送视频链接即可触发此技能。
